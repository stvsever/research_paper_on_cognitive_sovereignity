\begin{abstract}
\noindent
We present an ontology-constrained multi-agent simulation pipeline for auditing individual-difference moderators of adversarial opinion effectivity. The design is a full factorial over 25 deterministic pseudoprofiles, four cognitive-warfare attack mechanisms (Misleading Narrative Framing, Astroturf Comment Wave, Fear Appeal Scapegoating, LLM Chatbot Personalized Persuasion), and ten political-opinion leaves sampled across four policy domains, yielding 1{,}000 attacked scenarios. Profile features are restricted to three survey-mappable dimensions: Big Five personality (five domain-level means derived from NEO-PI-R facet structure), chronological age, and sex. This parsimonious three-block design produces eight profile-level predictors that are both statistically tractable at $N_p = 25$ and directly interpretable against ESS, Eurobarometer, and ANES instruments.

The analysis stack combines repeated-outcome structural equation modeling (SEM), task-specific ridge regression with reliability-weighted conditional susceptibility scoring, bootstrap rank uncertainty ($B = 200$), hierarchical feature importance, and a profile feature network computed from the deterministic profile matrix. Execution quality is monitored explicitly: the present implementation used deterministic fallback outputs for opinion-generation stages following API-credit exhaustion, so directional effectivity is near-random and moderator estimates are noise. That degraded execution does not invalidate the methodological contribution; it validates the platform's self-restraint: under fallback conditions, predictive fit collapses ($\mathrm{ICC}(1)=0.011$, ridge $\mathrm{CV}\text{-}R^2 = -0.175$, elastic net selects zero features), while the deterministic profile feature network remains fully interpretable (91 nodes, 1{,}807 Spearman edges, density $=.441$, $Q=.313$). The operational conclusion is that the pipeline correctly distinguishes execution-sensitive and execution-invariant components. A replication with live API opinion data and the same three-block profile design will allow direct evaluation of Big Five, age, and sex as susceptibility moderators under genuine adversarial scenario conditions.
\end{abstract}

\section{Introduction}

Online influence operations exploit cognitive shortcuts, affective resonance, and identity-congruent framing to shift political opinions without relying on technical intrusion \citep{bennett2018disinformation,lazer2018science,lewandowsky2017posttruth}. From a defensive standpoint, the operationally useful question is not only whether manipulative content circulates, but which individuals are more or less likely to shift opinion under adversarially designed exposure. Psychographic tailoring has made this question practically salient: messages aligned to personality structure can substantially outperform untailored variants \citep{matz2017psychological}, and resilience programmes are increasingly designed around cognitive characteristics \citep{kozyreva2020citizens,roozenbeek2022psychological,pennycook2021psychology}.

Personality — specifically the Big Five traits — occupies a central position in that literature. Conscientiousness is associated with deliberative processing and resistance to persuasion shortcuts; Neuroticism correlates with reactivity to emotionally loaded or fear-based appeals; Openness predicts receptivity to novel framings; Agreeableness relates to social conformity and compliance with authority cues; Extraversion influences susceptibility to social-proof and bandwagon mechanisms \citep{pennycook2019lazy,pennycook2021psychology}. Age and sex are standard demographic covariates in persuasion research: digital-media literacy increases through middle adulthood and declines in older age groups, and sex-differentiated responses to specific attack families have been documented in experimental settings \citep{kozyreva2020citizens}.

The present study assembles these variables into an auditable, reproducible simulation pipeline. The design encodes profile features, attack mechanisms, and political opinion leaves as hierarchical JSON ontologies. A full factorial crosses all three dimensions — producing a profile panel of attacked scenarios in which directional effectivity is estimable. The broader goal is a transparent measurement object that can be validated against human data using standard criterion-validity tests. This paper reports the pipeline's behavior under a deliberate stress test: an execution environment without live API opinion scoring, which reveals which analytical components remain interpretable and which correctly degrade.

\section{Methods}

\subsection{Study Design and Profile Ontology}

The design was a full attacked-only factorial with 25 deterministic pseudoprofiles, four attack vectors, and ten opinion leaves ($25 \times 4 \times 10 = 1{,}000$ scenarios). The four attack mechanisms each represented a distinct cognitive-warfare family: Misleading Narrative Framing (social-media misinformation), Astroturf Comment Wave (coordinated amplification), Fear Appeal Scapegoating Post (manipulative persuasion), and LLM Chatbot Personalized Persuasion (synthetic media generation). Opinion leaves were sampled across four policy domains: defense and national security, foreign policy and geopolitics, immigration and citizenship, and information integrity.

Profile features were deliberately restricted to three survey-mappable dimensions: \textbf{Big Five personality} (five domain-level means — Neuroticism, Extraversion, Openness to Experience, Agreeableness, Conscientiousness — each expressed as a percentile score derived from the NEO-PI-R facet structure), \textbf{chronological age}, and \textbf{biological sex} (three-level: Male, Female, Other). Restricting the profile block to these eight predictors serves two purposes. Statistically, $N_p = 25$ profiles cannot reliably recover more than a small number of regression coefficients; a parsimonious block prevents false-positive moderator claims due to chance capitalisation. Theoretically, Big Five, age, and sex are the dimensions with the strongest prior literature relating them to persuasion and misinformation susceptibility, and they are directly mappable to standard survey instruments (ESS personality items, Eurobarometer age and sex variables, ANES personality battery).

Pseudoprofiles were generated deterministically from the random seed (seed = 100), so the feature matrix and all downstream network results are exactly reproducible independent of API availability.

\subsection{Outcome Definition and Adversarial Effectivity}

For each attack-opinion-profile triple, the primary outcome was adversarial effectivity:
\begin{equation}
\mathrm{AE}_{iak} = (\mathrm{post}_{iak} - \mathrm{baseline}_{iak}) \times d_k,
\end{equation}
where $d_k \in \{-1,+1\}$ encodes the adversarial direction assigned to opinion leaf $k$. Positive $\mathrm{AE}$ indicates movement in the adversarially intended direction; negative values indicate resistance or backfire. Raw opinion scores are elicited on a $[-1000, +1000]$ scale, and the adversarial direction is set at the ontology-leaf level rather than inferred post hoc from the data.

\subsection{Analysis Stack}

The analysis combined five complementary layers. \textbf{Variance partitioning}: $\mathrm{ICC}(1)$ decomposed total variance in $|\Delta|$ into between-profile and within-profile components \citep{shrout1979intraclass}. \textbf{Structural equation modelling}: A repeated-outcome path SEM regressed ten adversarial-effectivity indicators on the eight profile moderators (five Big Five domain means, age, sex Female, sex Other), treated as a small-panel descriptive diagnostic \citep{rosseel2012lavaan}. \textbf{Conditional susceptibility}: Task-specific ridge models were fit for each attack-opinion cell; reliability-weighted aggregation produced a conditional susceptibility score per profile. \textbf{Bootstrap uncertainty}: Profile-level resampling ($B = 200$) quantified rank instability; false-discovery-rate control used the \citet{benjamini1995controlling} step-up procedure. \textbf{Profile feature network}: Spearman rank correlations between all profile features were thresholded at $|\rho| \ge .15$ to construct a weighted undirected graph. Community detection used a greedy modularity algorithm, and centrality metrics included degree, weighted strength, eigenvector centrality, PageRank, betweenness, closeness, participation coefficient, bridge ratio, within-module $z$-score, and $k$-core index.

\subsection{Advanced Profile Feature Network}

The network analysis is designed specifically for a mixed hierarchical panel in which continuous trait scales (Big Five domain means, age) coexist with categorical indicators (sex dummy variables). Node shapes distinguish feature types: circles for continuous features, diamonds for categorical-dummy nodes. Edge weights use $|\rho|$; path distances use $1/|\rho|$. Community convex hulls demarcate detected clusters in the interactive explorer; force-directed layout enables manual spreading of dense regions; lasso multi-selection supports cohort-level metric comparison. Bridge-oriented metrics — participation coefficient and bridge ratio — are particularly informative in this mixed design because they identify variables that link otherwise separated personality, age, and sex blocks. High degree alone does not reveal whether a node is a local hub within one block or a structural bridge between them; bridge diagnostics provide that distinction explicitly.

\subsection{Execution-Integrity Monitoring}

Execution quality is monitored as a first-class analytical layer. Baseline and post-exposure stages record fallback usage rates, realism review outcomes, and coherence check results. If opinion-generation stages fail, the analysis surfacing that failure and degrading inferential claims accordingly is the correct and expected behavior — not a limitation to be suppressed in reporting. This design principle is especially important in adversarial-simulation work, where peer reviewers are rightly skeptical of polished visualisations that mask methodological failure in the data-generation chain.

\section{Results}

\subsection{Study Design and Execution Integrity}

Figure~\ref{fig:design} summarises the full factorial structure. The present implementation completed all 1{,}000 scenarios structurally but used deterministic fallback outputs for opinion-generation throughout after API-credit exhaustion. Mean absolute displacement across all scenarios was 34.23 opinion-scale points; mean adversarial effectivity was 0.10; 49.6\% of rows showed positive adversarial effectivity. These metrics are consistent with essentially random directional success rather than genuine profile-driven opinion change. Figure~\ref{fig:matrix} displays mean AE across all attack $\times$ opinion cells, confirming the absence of systematic cell-level patterning under fallback conditions.

\subsection{Variance Partitioning and Predictive Diagnostics}

$\mathrm{ICC}(1)$ for absolute shift was 0.011, indicating that approximately 1.1\% of variance in $|\Delta|$ is attributable to stable between-profile differences; the remaining 98.9\% sits within profiles across tasks. Ridge cross-validated $R^2$ was $-0.175 \pm 0.203$; random-forest out-of-bag $R^2$ was $-0.194$; elastic net selected zero features. These results are expected under fallback-generated opinion data and confirm that the analysis stack correctly refuses to manufacture predictive structure when the outcome signal is absent.

\subsection{Structural Equation Model}

The repeated-outcome SEM converged with CFI $= 1.000$, RMSEA $= 0.000$, GFI $= .9995$, TLI $= 1.000$, and $n_{\mathrm{obs}} = 25$ profiles. At this sample size and under fallback-driven outcomes, near-perfect global fit is expected rather than informative — the model is substantially saturated. Figure~\ref{fig:moderation} displays the full moderator coefficient matrix across all five Big Five traits and sex, estimated via per-leaf OLS regression on the 25-profile panel.

Among the substantive paths, sex-linked effects were most prominent: Sex~Female $\to$ Humanitarian Intervention Support ($\hat\beta = 17.1$, $p < .05$), Sex~Female $\to$ Sanctions Use Support ($\hat\beta = 19.4$, $p < .05$), and Sex~Female $\to$ Media Trust ($\hat\beta = 22.4$, $p < .05$). Personality effects were consistent with theoretical expectations: Conscientiousness $\to$ Alliance Commitment Support ($\hat\beta = -9.4$, $p < .10$, indicating higher Conscientiousness predicts less fallback-driven movement toward that target) and Conscientiousness $\to$ Media Trust ($\hat\beta = 8.4$, $p < .05$). Neuroticism showed topic-specific reactivity (Humanitarian Intervention $\hat\beta = 6.9$, $p < .10$; Family Reunification $\hat\beta = -6.9$). Extraversion and Agreeableness produced weaker patterns with fewer significant paths. Given fallback data, all coefficient estimates should be treated as noise-sensitive artefacts rather than substantive findings. Their directional structure — Conscientiousness as a resistance moderator, Sex~Female and Neuroticism as amplifiers for specific opinion domains — is theoretically coherent but requires confirmation under live opinion scoring.

\subsection{Profile Feature Network}

The profile feature network contained 91 nodes and 1{,}807 edges at $|\rho| \ge .15$, with density $= .441$, five communities (greedy modularity $Q = .313$), average clustering $= .165$, and transitivity $= .509$. The mean node degree was 39.7 (maximum 59), indicating a highly cross-connected feature space. Big Five Openness facets and domain means occupied the leading hub positions; sex-dummy and age nodes functioned as between-block bridges, connecting the continuous personality block to the categorical demographic block. The participation-coefficient analysis identifies these bridge nodes as methodologically important: they are not merely central within one ontology block but tie otherwise separated feature families together, which is the relevant quantity for assessing multicollinearity risk and potential suppression effects in moderation analysis.

\subsection{Task Reliability and Susceptibility Estimation}

The task reliability surface shows the distribution of reliability weights (proportional to $n/\mathrm{CV\text{-}MSE}$) across the attack-opinion factorial. Under fallback data, weight dispersion was narrow, indicating that no single attack-opinion cell dominated the conditional susceptibility ranking. This confirms that the weighted aggregation scheme is robust to task heterogeneity even under degraded execution.

\subsection{Analytical Self-Restraint as the Correct Outcome}

The convergent failure of predictive diagnostics — negative $R^2$ across three estimators, near-zero ICC, zero elastic-net selections — is not a limitation but a validation. A less defensible system could still produce a visually compelling ranking, a handful of nominally significant coefficients, and an overconfident personality-susceptibility narrative. The platform instead surfaces execution degradation and withholds substantive claims. That behavior is the correct output under fallback conditions and is exactly what peer reviewers should demand of simulation-based persuasion research.

\section{Discussion}

\subsection{Big Five as the Primary Theoretical Frame}

The three-block profile design — personality, age, sex — is not merely a statistical convenience. It reflects a principled theoretical claim: among individual-difference predictors of adversarial opinion susceptibility, Big Five personality traits have the strongest prior evidence base and the most direct mechanistic links to cognitive-warfare attack families. Conscientiousness predicts deliberative processing and resistance to heuristic-based appeals \citep{pennycook2019lazy}. Neuroticism amplifies reactivity to fear and threat-based content \citep{lewandowsky2017posttruth}. Openness correlates with information seeking and exposure breadth, which may increase susceptibility to sophisticated narratives while reducing susceptibility to crude misinformation. Agreeableness links to social trust and authority compliance, making it relevant to astroturf and consensus-manufacturing attacks. Extraversion is associated with social information sharing and susceptibility to social-proof framing.

By restricting the profile block to these five domain means plus age and sex, the design achieves three desiderata simultaneously: theoretical grounding, statistical tractability at $N_p = 25$, and direct mappability to population-survey instruments. The latter is critical if simulation findings are to be validated against real survey data at a later stage.

\subsection{Methodological Lessons for the Field}

Three reviewer-facing observations follow directly from the present results. \emph{First}, no substantive claim about personality or sex differences in adversarial susceptibility is supported by this implementation. The fallback-dominated opinion stages invalidate that inference. \emph{Second}, the profile feature network should not be misread as evidence about persuasion outcomes: it characterizes the geometry of the modeled feature panel, providing a pre-modeling diagnostic of redundancy, bridge structure, and mixed-feature coupling. \emph{Third}, near-perfect SEM fit ($\mathrm{CFI} = 1.000$, $\mathrm{RMSEA} = 0.000$) at $N_p = 25$ in a substantially saturated model is informationally vacuous; it should not be cited as evidence of model quality.

The broader lesson is that synthetic susceptibility research requires more stringent self-assessment than observational studies, not less. The present platform is designed to enforce that standard: execution diagnostics, predictive-fit collapse, and inferential downgrading are first-class analytical outputs rather than footnotes. That architectural commitment is the primary contribution of this study.

\subsection{Path to Substantive Findings}

A replication with live API opinion data using the same three-block profile design is the next scientifically meaningful step. If that replication returns stable positive $\mathrm{CV}\text{-}R^2$ and identifiable personality moderators, the upgraded analytical stack and interactive diagnostics are already in place to present those findings defensibly. If predictive fit remains near zero under live data, the correct conclusion is that the four-attack, ten-opinion factorial surface contains weak profile-level signal — also a useful and informative result that constrains future theoretical work.

\section{Conclusion}

The present study demonstrates that an ontology-constrained multi-agent simulation pipeline can maintain analytical self-restraint under degraded execution while still delivering interpretable deterministic structure from the profile feature space. Three specific contributions follow:
\begin{enumerate}
  \item A parsimonious three-block profile design — Big Five personality domain means, age, and sex — that is statistically tractable at $N_p = 25$ and directly mappable to standard social-science survey instruments;
  \item An advanced profile feature network with bridge-sensitive centrality metrics that diagnoses hub-vs-bridge structure in a mixed continuous-plus-categorical feature panel, providing a pre-moderation assessment of redundancy and cross-block coupling; and
  \item An integrated execution-integrity monitoring layer that correctly surfaces diagnostic collapse and prevents unwarranted moderator claims when opinion-generation stages fail.
\end{enumerate}

These properties together define the standard that a simulation-based persuasion-auditing platform should meet before substantive susceptibility claims are advanced. The logical next step is to restore live opinion-generation and evaluate whether the three-block design recovers stable, theoretically interpretable personality, age, and sex effects under genuine adversarial-scenario conditions.

\section*{Dual-Use Ethics Statement}

This project models mechanisms that could, in principle, be repurposed for adversarial targeting. The intended use is defensive: auditing vulnerability structure, stress-testing analytical claims, and identifying where resilience interventions may be most needed. No real individuals are modeled; all profiles are synthetic; and the present implementation yields no operationally actionable targeting signal. Any future deployment on real data would require ethics review, access controls, and explicit prohibition of offensive use.

\section*{Reproducibility Statement}

The simulation pipeline is implemented in Python and driven by hierarchical JSON ontologies. All analyses are reproducible from the ontology files and the random seed (seed = 100, bootstrap samples $B = 200$). Stage-level artifacts — delta summaries, ICC outputs, SEM fit indices, ridge reliability weights, bootstrap CSVs, and network centrality files — are stored under \texttt{stage\_outputs/} with per-stage \texttt{manifest.json} files. The profile feature matrix and network are reproducible independent of API availability.
